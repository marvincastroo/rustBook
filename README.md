
# rustBook

Opening documentation
```
> rustup doc
```
Compiling and executing
```
> rustc main.rs
> .\main
```
## Cargo
Creating new
```
cargo new name
```
Converting to cargo
1. Move code to `/src` directory
2.  Run
```
cargo init
```
Running
```
cargo build
.\target\debug\hello_cargo.exe
```
or `cargo run` to do both in one

Check if it compiles
```
cargo check
```
